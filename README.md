# Day Planner Dashboard

A task management web app built with React, Redux, and Vite.

## Features
- Login & Signup with form validation
- Add, edit, delete tasks
- Mark tasks as done
- Drag and drop to reorder tasks
- Pagination

## Tech Stack
- React 19
- Redux Toolkit
- React Router DOM
- Formik
- Vite
